# 出力サンプル

SKILL.md に記載のない補足的な出力例を示す。
標準・簡易・詳細・表記統一のみモードの基本フォーマットは SKILL.md を参照。

---

## サンプル入力

以下のテキストを校正する場合を例とする。

```
弊社のサービスについてご説明させて頂きます。

このシステムは、ユーザが簡単に設定を見れる様に設計されています。以外と知られていませんが、
データーベースへのアクセスも可能です。

まず最初に、基本的な使い方を説明します。設定画面を開いて頂き、必要な項目を入力して下さい。
入力が完了したら、保存ボタンを押して頂ければと思います。

ご不明な点がございましたら、お気軽にお問い合わせ下さい。担当者が対応させて頂きます。
```

## 小説の場合の出力例

文書タイプが小説の場合、会話文の口語表現やキャラクターの口調を尊重する判断が必要。

### 入力

```
「明日、会えれる？」
彼女はそう言って、僕の方を見た。
「うん、会えるよ」
僕は頷いた。全然問題ない。むしろ嬉しいくらいだ。
```

### 出力（標準モード）

```markdown
## 校正結果

### 修正提案（確信度：高）

（なし）

### 要確認（確信度：低）※標準モードでは参考表示

| 行 | 表記 | 確認事項 |
|---|---|---|
| L1 | 会えれる | れ足す言葉。会話文のため口語表現として意図的であれば問題なし |
| L4 | 全然問題ない | 「全然」+肯定。現代口語では許容されるが、地の文のため確認 |

### コメント

会話文と地の文で文体が適切に使い分けられています。
「会えれる」は文法的には「会える」が正しいですが、キャラクターの口調として意図的であれば修正不要です。

※確信度：高の指摘がないため、修正版は出力しません（提案のみ）。
```

## 小説の場合の出力例（地の文に問題があるケース）

会話文は修正せず、地の文のみ修正する判断の違いを示す。

### 入力

```
「全然大丈夫だって。気にしないでよ」
彼女は笑って見せたが、その表情は以外にも硬かった。
僕は暫くの間、何も言えないまま、彼女の横顔を見つめていた。
時間が立つにつれ、彼女の肩の力が抜けていくのが分かった。
「ありがとう。ここに居てくれるだけで嬉しい」
```

### 出力（標準モード）

```markdown
## 校正結果

### 修正提案（確信度：高）

| 行 | 現在の表記 | 修正案 | 理由 |
|---|---|---|---|
| L2 | 以外にも | 意外にも | 同音異義語（「思いがけず」の意味） |
| L4 | 時間が立つ | 時間が経つ | 同音異義語（時間の経過は「経つ」） |

### 確認事項（確信度：中）

| 表記A | 出現 | 表記B | 出現 | 推奨 |
|---|---|---|---|---|
| 暫く | 1箇所 | （他に用例なし） | — | 「しばらく」（ひらがな推奨） |

### 要確認（確信度：低）※標準モードでは参考表示

| 行 | 表記 | 確認事項 |
|---|---|---|
| L1 | 「全然大丈夫」 | 会話文中の口語表現。修正不要 |
| L3 | 暫く | 小説では漢字表記も許容。ひらがなの方が読みやすい |
| L5 | 居てくれる | 会話文中の補助動詞。「いてくれる」が本則だが会話文として自然 |

### コメント

会話文の口語表現（「全然大丈夫」「居てくれる」）はキャラクターの自然な口調として維持しています。
地の文の同音異義語2箇所（確信度：高）を修正しました。表記揺れ（確信度：中）は提案のみで、修正版には反映していません。

---

## 修正版

「全然大丈夫だって。気にしないでよ」
彼女は笑って見せたが、その表情は意外にも硬かった。

```

## 確信度：中のみの出力例（修正版なし）

### 入力

```
このガイドでは、ユーザーの設定方法を説明します。
別ページでは、ユーザ向けFAQも公開しています。
```

### 出力（標準モード）

```markdown
## 校正結果

### 修正提案（確信度：高）

（なし）

### 確認事項（確信度：中）

| 表記A | 出現 | 表記B | 出現 | 推奨 |
|---|---|---|---|---|
| ユーザー | 1箇所 | ユーザ | 1箇所 | どちらかに統一 |

※確信度：高の指摘がないため、修正版は出力しません（提案のみ）。
```
