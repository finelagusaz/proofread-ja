# 表記統一ルール

## 漢字/ひらがな選択

### 基本方針

- 小説: 柔らかい印象を優先 → ひらがな多め
- 技術文書: 正確さを優先 → 漢字多め

### 判断が分かれる語

| 語 | 小説推奨 | 技術文書推奨 | 備考 |
|---|---|---|---|
| 事/こと | こと | 事 | 名詞化は「こと」優先 |
| 時/とき | とき | 時 | |
| 物/もの | もの | 物 | 抽象概念は「もの」 |
| 等/など | など | 等 | |

## カタカナ長音符

### 確認項目

- 全体で統一されているか
- 業界標準に従っているか

### 主な語例

| 長音あり | 長音なし | 備考 |
|---|---|---|
| コンピューター | コンピュータ | IT業界は長音なし多い |
| サーバー | サーバ | 同上 |
| ユーザー | ユーザ | 同上 |
| データベース | データベース | 「ス」は長音扱いしない |
