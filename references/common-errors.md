# よくある誤字脱字パターン

## 同音異義語

以下は特に見落としやすい同音異義語のリスト。このリストに載っていない同音異義語も、文脈から誤用と判断できる場合はすべて指摘すること。

### 紛らわしい同音異義語（頻出）

| 誤 | 正 | 文脈・用例 |
|---|---|---|
| 以外 | 意外 | 「意外と」「意外にも」「意外な結果」 |
| 意外 | 以外 | 「〜以外」「それ以外」「君以外」 |
| 早い | 速い | 「時間が早い」は時刻、「速度が速い」はスピード |
| 立つ | 経つ | 「時間が経つ」「年月が経つ」 |
| 関わる | 係わる | 「仕事に関わる」「事件に関わる」※一般的な関与は「関」 |
| 効く | 利く | 「気が利く」「融通が利く」「小回りが利く」 |
| 計る | 測る/量る/図る/諮る | 測定は「測」、重さは「量」、企図は「図」、意見を求めるは「諮」 |
| 進める | 勧める/薦める | 勧誘は「勧」、推薦は「薦」 |
| 勤める | 務める/努める | 任務は「務」、努力は「努」 |
| 捕らえる | 捉える | 「本質を捉える」は抽象的 |

### ひらがな表記推奨（形式名詞・補助動詞）

| 漢字表記 | ひらがな表記 | 使い分けの基準 |
|---|---|---|
| 〜する事 | 〜すること | 形式名詞はひらがな（「大事な事」など実質名詞は漢字可） |
| 〜した時 | 〜したとき | 形式名詞はひらがな（「その時」など時点を指す場合は漢字可） |
| 〜する様に | 〜するように | 形式名詞はひらがな |
| 〜する為に | 〜するために | 形式名詞はひらがな |
| 〜と言う | 〜という | 形式動詞はひらがな（「〜と言う」が動作の場合は漢字可） |
| 〜に於いて | 〜において | ひらがな推奨 |
| 〜に取って | 〜にとって | 形式助詞はひらがな |
| 〜に因って | 〜によって | 形式助詞はひらがな |
| 〜に付いて | 〜について | 形式助詞はひらがな（「壁に付いて」など物理的付着は漢字） |
| 居る | いる | 補助動詞は「いる」（〜している） |
| 有る/在る | ある | 補助動詞は「ある」（〜してある） |
| 見る | みる | 補助動詞は「みる」（〜してみる） |
| 行く | いく | 補助動詞は「いく」（〜していく） |
| 出来る | できる | 可能の意味は「できる」 |
| 貰う | もらう | 補助動詞は「もらう」（〜してもらう） |

## 意味の取り違え（本来の意味と異なる用法で使われやすい語）

表現自体は正しいが、意味を取り違えて使われることが多い。誤った意味で使用されている場合に指摘する。

| 表現 | よくある誤解 | 本来の意味 |
|---|---|---|
| 役不足 | 能力不足の意味で使用 | 能力に対して役目が軽すぎること |
| 気が置けない | 油断ならないの意味で使用 | 遠慮がいらない、気楽に付き合える |
| 煮詰まる | 行き詰まるの意味で使用 | 議論が十分になされ結論に近づくこと |
| 姑息 | 卑怯の意味で使用 | 一時しのぎ、その場逃れ |
| 檄を飛ばす | 叱咤激励の意味で使用 | 自分の主張を広める文書を送ること |
| 確信犯 | わざとやった人の意味で使用 | 信念に基づいて行う犯罪（政治犯・思想犯） |
| 小春日和 | 春の暖かい日の意味で使用 | 晩秋〜初冬の暖かく穏やかな天気 |
| 情けは人の為ならず | 甘やかすなの意味で使用 | 人に情けをかけると巡り巡って自分に返る |

## 尊敬語・謙譲語の誤用

| 誤 | 正 | 解説 |
|---|---|---|
| お伺いします | 伺います | 「伺う」自体が謙譲語（二重敬語） |
| 拝見させていただく | 拝見する | 二重敬語 |
| おっしゃられる | おっしゃる | 二重敬語 |
| 参られる | いらっしゃる/来られる | 謙譲語を尊敬語に誤用 |
| 申される | おっしゃる/言われる | 謙譲語を尊敬語に誤用 |

## 重複表現（二重表現）

同じ意味の語が重なった冗長な表現。

| 誤（重複） | 正 | 重複する要素 |
|---|---|---|
| まず最初に | まず / 最初に | 「まず」＝「最初」 |
| 後で後悔する | 後悔する | 「後悔」に「後で」が含まれる |
| 頭痛が痛い | 頭が痛い / 頭痛がする | 「頭痛」に「痛い」が含まれる |
| 一番最初 | 最初 / 一番先 | 「一番」＝「最」 |
| 過半数を超える | 半数を超える | 「過」に「超える」が含まれる |
| 約〜ぐらい | 約〜 / 〜ぐらい | 「約」＝「ぐらい」 |
| 違和感を感じる | 違和感がある / 違和を感じる | 「感」が重複 |
| 犯罪を犯す | 罪を犯す / 犯罪を行う | 「犯」が重複 |
| 日本に来日する | 来日する / 日本に来る | 「来日」に「日本に来る」が含まれる |

## ら抜き言葉

### 判別方法

「〜ない」を付けて判別する：
- 「見ない」→「み」で終わる → 上一段 →「見られる」が正
- 「食べない」→「べ」で終わる → 下一段 →「食べられる」が正
- 「書かない」→「か」で終わる → 五段 →「書ける」でOK（ら抜きではない）

書き言葉では避ける。会話文では確信度：低とする。

## さ入れ言葉

不要な「さ」が挿入されたもの。五段動詞の使役形で発生する。

| 誤（さ入れ） | 正 |
|---|---|
| 読まさせていただく | 読ませていただく |
| 書かさせていただく | 書かせていただく |
| 行かさせていただく | 行かせていただく |

## れ足す言葉

不要な「れ」が挿入されたもの。五段動詞の可能形に「れ」は不要。

| 誤（れ足す） | 正 |
|---|---|
| 行けれる | 行ける |
| 書けれる | 書ける |
| 読めれる | 読める |

## その他の冗長表現

| 冗長表現 | 簡潔な表現 | 解説 |
|---|---|---|
| 〜になります | 〜です | 「なります」は変化を表す |
| させていただく（多用） | する/いたします | 過剰な敬語 |
| 〜的な（多用） | 〜な/〜の | 「的」の多用を避ける |

## カタカナ表記の誤り

| 誤 | 正 | 理由 |
|---|---|---|
| シュミレーション | シミュレーション | simulationの発音 |
| ディフォルト | デフォルト | defaultの発音 |
| データーベース | データベース | 「ー」不要 |

## 間違いやすいが正しい表記

誤りとして指摘されやすいが、実際には正しい表記。誤検出を防ぐための参照。

| 正しい表記 | よくある誤解 | 解説 |
|---|---|---|
| 全然良い（口語） | 全然は否定のみ？ | 現代口語では肯定も可（文章では避ける） |
| やむを得ない | 「已むを得ない」では？ | 「止む」が正しい |
